# Camera2 Training

## Training for using the Camera2 API

Used the [this repository](https://github.com/googlesamples/android-Camera2Basic) as reference.

### Step 1

Connected to the camera, and displayed in the Texture view.
In this step, preview does not appear in some device.

### Step 2

Corresponding to the orientation of the device.

### Step 3

Implement the still image shooting function.

### Step 4

Implement the save function of still image.

### Step 5

Implement the switching function of the back and front camera.
